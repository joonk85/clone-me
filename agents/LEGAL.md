## 이용약관 핵심 조항

```
제1조 (목적)
clone.me는 전문가(마스터)와 이용자(멤버)를 AI 클론을 통해 연결하는 플랫폼입니다.

제5조 (토큰 정책)
- 구매한 토큰은 환불되지 않습니다 (서비스 오류 제외)
- 보너스 토큰은 지급일로부터 30일 후 자동 소멸합니다
- 무료 체험은 클론당 기본 5회, 설문 완료 시 +10회 추가됩니다

제8조 (마스터 의무)
- 본인 소유 자료만 업로드 가능합니다
- 타인 저작물 무단 업로드 시 즉시 서비스 중단됩니다
- 허위 이력/자격 등록 시 즉시 서비스 중단됩니다

제9조 (클론 자료)
- 업로드된 자료는 AI 학습에 활용됩니다
- 원본 파일은 출처 표시 정확도를 위해 서버에 보관됩니다
- 마스터 탈퇴 시 클론은 비활성화, 대화이력은 90일 후 삭제됩니다

제10조 (AI 면책)
- AI 클론의 답변은 참고용이며 전문적 조언을 대체하지 않습니다
- 플랫폼은 AI 답변의 정확성을 보증하지 않습니다
```

## RAG 관련 법적 고려사항

### 저작권
- 마스터 업로드 자료: 본인 소유 자료만 허용
- 타인 저작물 (책 전체, 강의 영상 등) 무단 업로드 금지
- 이용약관에 명시 필수

### 개인정보 (대화 내용)
- 대화 내용 익명화 저장 가능 (security_anonymize_conversations)
- RAG 청크에는 대화 내용 포함 안 됨 (학습 자료만)
- 멤버 인구통계 (설문): 수집 동의 후 익명으로만 활용

### AI 출처 표시
- "이 답변은 [파일명]에서 나왔습니다" 표시 = 투명성 확보
- 할루시네이션 방지 설정 (quality_no_answer=true) 권장

## 설정 페이지 동의 항목

```

대화 알림: 기본 ON

```

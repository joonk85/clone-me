# 🎭 Brand Agent — Brand Designer
> 항상 BRAIN.md를 먼저 읽으세요.

## 역할
브랜드 아이덴티티, 비주얼 언어, 마케팅 카피를 담당합니다.

## 브랜드 에센스
**"내 지식의 클론을 만드세요"**
전문가의 지식이 시간과 공간을 초월해 누구에게나 닿는 플랫폼

## 브랜드 퍼스낼리티
- **Expert** — 전문적이고 신뢰할 수 있는
- **Accessible** — 누구나 쉽게 접근할 수 있는
- **Innovative** — 한국 최초, 새로운 방식
- **Human** — AI지만 사람의 온기가 느껴지는

## 톤 오브 보이스
- 격식체와 친근체의 중간 (합니다/해요 혼용)
- 기술 용어는 풀어서 설명
- 마스터를 영웅으로, 멤버를 주인공으로

## 비주얼 아이덴티티

### 로고
```
clone.me
- 소문자 영문, Syne 800
- 'c': 원 안 아이콘 (cyan 배경, 검정 c)
```

### 컬러
- Primary: Cyan (#63d9ff) — 기술적, 미래적
- Secondary: Green (#4fffb0) — 성장, 수익
- Accent: Gold (#ffc832) — 검증, 신뢰

### 그라디언트
```css
/* 히어로 텍스트 */
linear-gradient(135deg, #fff 0%, #63d9ff 50%, #4fffb0 100%)
/* 배경 글로우 */
radial-gradient(circle, rgba(99,217,255,0.1) 0%, transparent 70%)
```

## 카피라이팅

### 마스터 대상
- "자는 동안에도 클론이 가르치고 수익이 입금됩니다"
- "내 어투, 내 경험, 내 지식 그대로 — AI 클론"
- "SRT 자막 하나로 내 말투가 살아납니다"

### 멤버 대상
- "분야 최고 전문가에게 24시간 1:1로 물어보세요"
- "전문가 클론이 항상 당신 곁에"

### 출처 표시 관련
- "어느 강의 어디에서 나온 답인지 정확히 알려드립니다"
- "🔐 암호화 문서 기반 답변"

### 클론 품질 관련
- "더 많은 자료 = 더 나다운 클론"
- "SRT 자막 한 개가 어투를 살립니다"

### 베타 뱃지
```
지식 클론 플랫폼 · 한국 최초 · BETA
```

## 마케팅 메시지 (클론 출처 기능)
출처 표시 기능은 단순한 투명성이 아니라 마케팅 채널입니다:
"📺 콜드콜 강의 3분42초에서 나온 내용입니다 → [강의 보기]"
→ 클론 대화가 자연스러운 강의/상품 추천으로 연결됩니다.

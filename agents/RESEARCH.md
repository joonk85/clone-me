# 🔬 Research Agent
> 항상 BRAIN.md를 먼저 읽으세요.

## 역할
시장 조사, 경쟁사 분석, 유저 인사이트, 클론 품질 전략을 담당합니다.

## 시장 현황

### 경쟁사
| 서비스 | 특징 | 약점 |
|---|---|---|
| Delphi AI (글로벌) | AI 클론 선구자 | 한국 미진출, 영어 중심 |
| 인프런 | 한국 1위 강의 | 1:1 대화 없음 |
| 클래스101 | 크리에이터 중심 | AI 없음 |
| 탈잉 | 튜터 매칭 | 비용 높음, 스케일 한계 |

### 우리의 포지션
- 한국 시장 직접 경쟁자 없음
- "강의 구매" → "전문가와 대화" 패러다임 전환
- 마스터 시간 한계 해결 = 진짜 문제 해결

## 클론 품질 전략 (핵심 인사이트)

### 왜 클론이 "그냥 ChatGPT 같다"는 느낌이 드는가
사람의 지식은 3층: 명시적(글)/암묵적(어투)/경험적(케이스)
기존 RAG는 명시적 지식만 학습 → 어투·경험이 없어서 밋밋함

### 해결책: 자료 카테고리 A~E
```
A. 어투/스타일 (SRT 자막, SNS) ← 가장 임팩트 큼
B. 핵심 지식 (강의 스크립트, PDF)
C. 케이스/경험 (구조화 템플릿으로 수집)
D. 판단 기준 (추천/비추천, 원칙)
E. 고정 답변 (AI보다 우선)
```

### 마스터 설득 포인트
- SRT 1개만 있어도 어투가 확연히 달라짐
- "내 자료가 많을수록 더 나처럼 대화"
- 자료별 참조 현황으로 어떤 자료가 핵심인지 시각화

### 출처 표시의 마케팅 효과
- "📺 콜드콜 강의 3분42초" → 해당 강의 구매 링크 연결
- 멤버가 흥미를 가진 주제 → 강의 자동 추천
- 클론이 마케팅 채널이 됨

## 핵심 인사이트

- 한국 유저 80% 모바일 접속 → 모바일 필수
- 강사 가장 큰 불안: "내 클론이 틀린 말 할까봐"
  → 해결: 런칭 전 10가지 테스트 + 원본 파일 출처 표시
- 멤버 설득 포인트: "월 2만원 = 유명 강사 24시간"
- 락인 전략: 대화 이력 + 자료 참조 데이터 = 이탈 불가

## 성공 지표 (MVP)
- 마스터 1명 온보딩 + 클론 품질 점수 75점 이상
- 구독자 10명
- 재구독률 40%+
- 이 데이터가 없으면 투자자 설득 불가

## 유저 페르소나

### 마스터
- 30~45세, 유튜브/SNS 팔로워 1만+
- 연 수익 있음, 시간 부족이 가장 큰 고민
- 기술 친화도: 보통
- 관심사: "내가 없어도 내 지식이 수익을 낼 수 있나?"

### 멤버
- 20~35세, 성장 욕구 강함
- 유료 강의 구매 경험 있음
- "바로 답 원함, 강의 영상 보기 귀찮음"
- 카카오페이, 토스 결제 익숙
